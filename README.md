# Word2Belt


# Installation

```bash
pip install git+https://github.com/mpkato/word2belt.git
```

# Usage
```
usage: word2belt [-h] --url URL --username USERNAME --password PASSWORD
                 --filename FILENAME
                 word

Script to vectorize a word

positional arguments:
  word                 Word to be vectorized

optional arguments:
  -h, --help           show this help message and exit
  --url URL            URL
  --username USERNAME  Username for Basic Auth
  --password PASSWORD  Password for Basic Auth
  --filename FILENAME  Filename of a Word2Vec model
```
